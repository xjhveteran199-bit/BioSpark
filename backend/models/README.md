# Pre-trained Models

Place ONNX model files in this directory.

## Available Models

| File | Task | Signal | Size |
|------|------|--------|------|
| `ecg_arrhythmia_cnn.onnx` | Arrhythmia Detection | ECG | ~400KB |
| `eeg_sleep_staging.onnx` | Sleep Staging | EEG | ~4MB |
| `emg_gesture_cnn.onnx` | Gesture Recognition | EMG | ~800KB |

## Download

Models will be auto-downloaded on first use, or you can manually place them here.
