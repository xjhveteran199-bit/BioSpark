## 一、项目概述

**BioSpark** 是一个面向生物信号（ECG心电、EEG脑电、EMG肌电）的端到端 AI 分析与训练平台。用户无需编程能力，只需上传数据即可完成信号分析、模型推理和自定义 CNN 训练。

**核心定位：** 降低生物信号深度学习的使用门槛，让医学研究者和工程师能快速验证算法思路。

**在线地址：**
- Railway (全功能): https://efficient-integrity-production-736e.up.railway.app
- Vercel (轻量前端): https://infallible-blackwell.vercel.app
- GitHub: https://github.com/xjhveteran199-bit/BioSpark

## 四、CNN 模型架构

### Signal1DCNN — 通用一维卷积神经网络

```
Input: (batch, n_channels, signal_length)
  ↓
Conv1d(n_channels, 32, kernel=7) → BatchNorm → ReLU → MaxPool(2)
  ↓
Conv1d(32, 64, kernel=5) → BatchNorm → ReLU → MaxPool(2)
  ↓
Conv1d(64, 128, kernel=3) → BatchNorm → ReLU → MaxPool(2)
  ↓
GlobalAveragePooling1d → 128维特征向量
  ↓
FC(128→64) → ReLU → Dropout(0.3)
  ↓
FC(64→n_classes) → Softmax
  ↓
Output: (batch, n_classes) 概率分布
```

**参数量：** ~44K（轻量级，CPU 可快速训练）

## 十一、总结

BioSpark 已完成从 0 到 1 的核心产品搭建，具备完整的信号分析和模型训练能力。当前平台支持 ECG 心律失常检测，并提供了灵活的自定义训练管线。

3. **实时可视化** — WebSocket 实时训练曲线，交互式图表
4. **灵活部署** — Docker/Railway/Vercel 三种部署模式
5. **开放架构** — 模型可插拔，格式可扩展

**下一步重点：** 补齐 EEG/EMG 模型 → 用户系统 → 模型市场 → 商业化。
