Data Format Specification

CSV Format (Recommended)

Each row = one sample. Must contain a label column.

Single-channel: 
s1,   s2,   s3,   ..., s140, label
0.12, 0.34, 0.56, ..., 0.78, Normal
0.11, 0.45, 0.67, ..., 0.89, Arrhythmia

All numeric columns (excluding label/time) are treated as one continuous signal.

Multi-channel: 
ch1_1, ch1_2, ..., ch1_70, ch2_1, ch2_2, ..., ch2_70, label
0.12,  0.34,  ..., 0.56,  0.78,  0.90,  ..., 0.11,  Normal

Use ch{N}_{sample} naming. All channels must have equal length.

ZIP Format

dataset.zip/
  Normal/
    sample_001.csv
    sample_002.csv
  Arrhythmia/
    sample_001.csv
    sample_002.csv

Folder name = class label. Each CSV = one sample.

Column Name Rules

Role Recognized Names Behavior
Label label, class, target, y Classification target (required)
Time time, t, timestamp, index, ms, sec Auto-excluded
Channel ch1_1, ch2_1, channel1_1, ... Auto-grouped into channels
Signal Any other numeric column Signal features for CNN
